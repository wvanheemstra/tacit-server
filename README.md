# tacit-server

Server module
==================

The server module for the tacit system.

NOTE: This is a submodule of tacit.

## Installation

  npm install tacit-server --save

## Usage

  var tacitServer = require('tacit-server'),
      server = tacitServer.server;

## Tests

  npm test
